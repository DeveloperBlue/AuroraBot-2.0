Just Another Discord Bot

AuroraBot is a Discord bot built with Node JS. It prides itself on having a small number of dependencies, and is currently locally hosted on a dedicated server. Anything from listening to music, rolling some dice, getting photos from NASA satelites at specific coordinates, or keeping activity logs, AuroraBot strives to deliver with as much functionality as possible!

Features

AuroraBot has a variety of useful, unique, and fun commands!

Fun!

AuroraBot has a ton of fun commands. Roll some dice, make it choose between things for you, get some pictures from a NASA rover or satellite, compute crazy queries with Wolfram, and look up Anime information and streaming sites.

Mostly Available

AuroraBot is hosted on a server in my basement. Therefore, things can go wrong. It's generally up 24/7, but this is here to cover my bases. Willing to pay for some AWS/Google Cloud dedicated Node servers? Contact me!

Moderation

AuroraBot handles moderation with ease. It is able to keep thorough logs for various events like message deletion, status changes, message editting, etc. and output them in whatever specific channels you decide.

Music

AuroraBot plays music too! With extreme functionality, add playlists, unqueue certain songs, adjust dedicated text or voice channels, and more!

Currently locked to YouTube, but support for 100+ websites (Including SoundCloud) are on the way!

Chatty

AuroraBot is always reading everything you say! EVERYTHING! #NSA #FBI #SONY #AlexaLaughsRandomlyAtNight. A chatbot API is on it's way, which will allow you to have some plain retardation level conversations with her, and it won't feel like a CleverBot rip off!

NSFW Channel Control

You can choose to completely remove and hide NSFW commands, have them work only in NSFW marked channels, only in white-listed channels, or in all channels but black-listed channels. You control where the NSFW goes.
