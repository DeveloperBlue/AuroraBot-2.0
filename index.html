Just Another Discord Bot

AuroraBot is a Discord bot built with Node JS. It prides itself on having a small number of dependencies, and is currently locally hosted on a dedicated server. Anything from listening to music, rolling some dice, getting photos from NASA satelites at specific coordinates, or keeping activity logs, AuroraBot strives to deliver with as much functionality as possible!

Features

AuroraBot has a variety of useful, unique, and fun commands!

Fun!

AuroraBot has a ton of fun commands. Anything from the generic rolling of dice and flipping a coin to browsing pictures of NASA rovers, getting what's trending on Twitter, compute crazy queries with Wolfram Alpha, and look up Anime information and streaming sites.

Mostly Available

AuroraBot is hosted on a dedicated server rig in my basement. Therefore, things can go wrong. It's generally up 24/7, but of course accidents and residential problems can occur. Aer you willing to pay for some AWS/Google Cloud dedicated Node JS servers? Feel free to contact me!

Moderation

AuroraBot handles moderation with ease. It is able to keep thorough logs for various events like message deletion, status changes, message editting, etc. and output them in whatever specific channels you would like. Also features an auto-role system for allowing users to assign themselves certain roles.

Music

Featuring extreme functionality, AuroraBot can add playlists, re-arrange, micro-manage, and shuffle the queue, set dedicated text and voice channels, and more!

AuroraBot supports over 100+ services, including YouTube, SoundCloud, and Vimeo. You can also play direct .mp3 links and uploaded .mp3 files!

Chatty

AuroraBot is always reading everything you say! EVERYTHING! #NSA #FBI #SONY #AlexaLaughsRandomlyAtNight. A chatbot API is on it's way, which can allow you to have some slightly-better-than-Cleverbot level conversations!

NSFW Channel Control

You can choose to completely remove and hide NSFW commands, have them work only in NSFW marked channels, only in white-listed channels, or in all channels but black-listed channels. You control where the NSFW goes. AuroraBot features a good number of NSFW commands, ranging from querying adult websites to triggering lewd neko bombs.
